# Roles y pantallas por rol

## Roles sugeridos

1. **Admin**
   - Gestión de usuarios y roles
   - Configuración de catálogos, precios y zonas
   - Visualización completa de reportes

2. **Gerencia / Supervisor**
   - Dashboard de ventas y cobranza por equipo
   - Seguimiento de cumplimiento por zona
   - Alertas por cartera vencida

3. **Vendedor**
   - Pantalla de ruta diaria
   - Registro de pedidos/ventas
   - Estado de clientes atendidos

4. **Cobrador**
   - Lista de cobros pendientes por ruta
   - Registro de pago (efectivo, transferencia)
   - Captura de geolocalización en cada cobro

5. **Consulta (solo lectura)**
   - Dashboard limitado
   - Reportes sin capacidad de edición

## Matriz de acceso (resumen)

- Usuarios/Roles: Admin
- Clientes: Admin, Supervisor, Vendedor
- Ventas: Admin, Supervisor, Vendedor
- Cobros: Admin, Supervisor, Cobrador
- Dashboard: Admin, Supervisor, Consulta
- Configuración global: Admin
